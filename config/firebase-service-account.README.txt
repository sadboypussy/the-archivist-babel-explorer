Fichier optionnel : firebase-service-account.json
===============================================

Pour publier des découvertes vers la galerie Firestore depuis l’application :

1. Dans la console Firebase : Paramètres du projet → Comptes de service →
   « Générer une nouvelle clé privée » (fichier .json).

2. Enregistrez ce fichier ICI, sous le nom exact :

   firebase-service-account.json

   (même dossier que ce fichier README.)

3. Relancez l’application (double-clic sur Launch-Archivist-UI.bat, ou Launch-Archivist-UI-headless.bat sans navigateur).

Ce fichier ne doit jamais être envoyé sur Git : il est listé dans .gitignore.

Alternative : n’importe quel chemin + variable d’environnement
ARCHIVIST_FIREBASE_CREDENTIALS (voir DISTRIBUTION.md).
